¿Que es @JasnahBot @JasnahBot?


JasnahBot es un Bot de telegram que utiliza la API Openlibray para obtener información de las peticones del usuario.


¿Que tipo de busquedas puedes hacer a JasnahBot?

Al entrar en el bot encontraras un teclado con las opciones que tiene y ejemplos de como usar cada uno de ellos

Busqueda General Libros
Para buscar por libros por titulo, saga o autor
Ej: /search Brandon Sanderson

Busqueda por isbn
Buscar un libro directamente mediante su isbn
Ej: /isbn 9780765350381

Por defecto, la API devuelve la respuesta en formato JavaScript.

La API de ISBN es un caso especial y un enfoque alternativo para llegar a una página de ediciones. En lugar de "/books", se utiliza una ruta de "/isbn", seguida de un ISBN 10 o 13 válido.

Aquí tienes un ejemplo:

https://openlibrary.org/isbn/9780140328721

En este ejemplo, ingresar esta URL resultará en una redirección a la página de ediciones correspondiente:

https://openlibrary.org/books/OL7353617M

Al igual que en una página de Edición o Trabajo, podemos agregar ".json" al final de la URL para solicitar la respuesta en formato json en lugar de HTML, por ejemplo:

https://openlibrary.org/isbn/9780140328721.json
